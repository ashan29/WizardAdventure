Concept du jeu:

Wizard Adventure est un jeu divertissant qui est jouable à un seul joueur. Le joueur a la possibilité de choisir un magicien parmi les trois proposés. Une fois que le personnage est choisi, le joueur peut commencer à jouer à Wizard Adventure. Le but du jeu est de sauver la princesse capturée par le big boss. Pour cela, le joueur doit être courageux et il doit entrer dans le château du big boss. Ensuite, il faut retrouver la princess, de la libérer et de quitter le château avec la princesse. Dans chaque niveau, le joueur est confronté à se battre avec un petit monstre. Au total, le jeu compte cinq niveaux différents. Le joueur a uniquement une seule vie. Chaque fois qu'il perd sa vie, le jeu sera terminé. Alors, est-ce que tu es assez courageux pour sauver la princess ?
