Concept du jeu:

Wizard Adventure est un jeu divertissant qui est jouable à un seul joueur. Le joueur a la possibilité de choisir un personnage parmi les trois proposés, qui sont Romain, Nelson et Ashan. Une fois que le personnage est choisi, le joueur peut commencer à jouer à Wizard Adventure. Le but du jeu est de combattre le boss qui se trouve dans le niveau. Si le joueur parvient à combattre ce boss. Il passera au prochain niveau et il pourra défier un autre boss plus difficile que le précédent. Au total, le jeu compte dix niveaux différents. Si le jouer atteint le niveau dix, il rencontrera le big boss et il doit le vaincre afin de réussir le jeu. Le joueur a uniquement trois vies. Chaque fois qu’il est battu par le boss, il reculera d’un niveau. S’il perd les trois vies, il perdra le jeu.
